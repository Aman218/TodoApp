# ToDoProject

This is a basic To-Do list web application built using **React** and **Vite**.

## Project Structure

```bash
ToDoProject/
├── node_modules/
├── public/
├── src/
│   ├── assets/
│   ├── component/
│   │   ├── Header.css
│   │   ├── Header.jsx
│   │   ├── TodoItem.css
│   │   ├── TodoItem.jsx
│   │   ├── TodoList.css
│   │   ├── TodoList.jsx
│   ├── App.css
│   ├── App.jsx
│   ├── index.css
│   ├── main.jsx
├── .gitignore
├── package-lock.json
├── package.json
├── vite.config.js
└── index.html

## Requirements

Before you begin, ensure you have the following installed:

i-Node.js (v14 or later)
  npm or yarn

ii-Navigate into the project directory i.e cd ToDoProject

iii-Install the dependencies: i.e npm install
# or if using yarn
yarn


iv-To run the development server with Vite: i.e npm run dev
